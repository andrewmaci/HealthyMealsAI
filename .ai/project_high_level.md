Application - HealthyMeal (MVP)
Core Problem
Adapting culinary recipes found online to personal nutritional needs and dietary requirements is problematic. The application uses AI and user preferences to suggest tailored recipes.

Minimum Viable Product (MVP) Functionality
Saving, reading, viewing, and deleting recipes in text format

A simple user account system to link a user with their own recipes

A user profile page for saving dietary preferences

AI integration to allow modification of recipes based on user's dietary preferences

What is NOT included in the MVP
Importing recipes from a URL

Rich multimedia support (e.g., recipe photos)

Sharing recipes with other users

Social features

Success Criteria
90% of users have filled in the dietary preferences section in their profile

75% of users generate one or more recipes per week
